# 出物小助手 - 全链路使用手册

## 产品概述

**出物小助手**是一款面向谷圈、小卡、娃圈、潮玩等收藏用户的微信小程序，帮助用户记录收藏、整理心愿单、生成出物清单图和多平台出物文案。

本手册覆盖完整的业务链路，供开发和测试验收使用。

## 一、核心业务链路总览

本小程序的完整业务链路分为 **4 条主要流程**：

### 链路 1：用户登录与注册

```
入口：小程序打开 → 启动页(登录页)
    ├── [微信一键登录] → 获取微信code → 后端创建/登录用户 → 返回JWT token → 进入首页
    └── [先随便看看] → 游客模式首页(不可保存数据)
        触发保存操作时 → 弹出登录引导
```

### 链路 2：收藏管理流程

```
首页/收藏Tab → [记一件谷子] → 新增收藏页
    ├── 上传图片(1-9张)
    ├── 填写名称(必填)、类型(必填)、角色、作品、入手价、状态等
    ├── [保存] → 收藏列表页
    └── [保存并继续添加] → 清空表单继续

收藏列表页
    ├── 搜索/筛选/排序 → 查看收藏
    ├── 点击卡片 → 收藏详情页 → 编辑/标记已出/加入出物清单
    ├── 批量模式 → 批量修改状态/删除
    └── [+] → 新增收藏页
```

### 链路 3：出物清单生成流程（核心传播链路）

```
首页/出物Tab → [生成出物清单]
    → 选择收藏品页(勾选要出的商品)
    → [下一步] → 编辑出物信息页
        ├── 填写每个商品的出物价/瑕疵/包邮规则
        ├── 设置交易规则(平台/是否拆售/联系方式)
        └── [保存草稿] | [下一步：选择模板]
    → 模板选择页(简洁表格款/卡片展示款/图片墙款)
    → [生成长图] → 生成结果页
        ├── [保存图片] → 相册
        ├── [分享给好友] → 微信分享
        ├── [复制文案] → 闲鱼/小红书/微博/微信群文案
        ├── [重新编辑] → 返回编辑页
        └── [换模板] → 返回模板选择
    → [分享] → 生成分享链接 → 他人打开分享页查看
```

### 链路 4：心愿单转收藏流程

```
首页/收藏页 → [添加心愿] → 新增心愿页
    ├── 填写名称(必填)、目标价、想买程度等
    ├── [保存] → 心愿单页
    ├── [保存并转为收藏] → 同时创建收藏记录
    └── 心愿单 → 标记已入 → 确认转为收藏 → 自动带入新增收藏页
```

## 二、页面功能详解

### 2.1 启动/登录页 (`pages/login/index`)

**路由**: `/pages/login/index`

**页面元素**:
- Logo 区域：产品图标 + 名称"出物小助手" + 标语
- 卖点展示：3 张卖点卡片
- 协议勾选：《用户协议》《隐私政策》
- 主按钮：微信一键登录
- 次入口：先随便看看

**交互流程**:
| 操作 | 流程 |
|------|------|
| 微信一键登录 | 勾选协议 → 调用 `wx.login` → `POST /auth/wechat-login` → 存储 token → 跳转首页 |
| 先随便看看 | 游客模式进入首页，无法保存数据，触发保存时弹登录引导 |
| 未勾选协议 | Toast 提示"请先阅读并同意用户协议和隐私政策" |
| 登录失败 | Toast 提示"登录失败，请稍后重试" |

**API 调用**:
- `POST /api/v1/auth/wechat-login` (code, nickname, avatar) → `{ token, refreshToken, user }`

**前端文件**:
- [login/index.tsx](miniprogram-demo/src/pages/login/index.tsx)

### 2.3 收藏列表页 (`pages/collections/index`) - Tab 页

**路由**: `/pages/collections/index`

**筛选标签**: 全部、已到货、预售、待补款、待发货、待出物、已出物、心愿已入

**排序方式**:
- 最近添加 (`created_at:desc`)
- 最近更新 (`updated_at:desc`)
- 入手价从高到低 (`price:desc`)
- 入手价从低到高 (`price:asc`)
- 名称 A-Z (`name:asc`)

**交互**:

| 操作 | 方式 | 结果 |
|------|------|------|
| 搜索 | 输入关键词实时过滤 | 搜索名称/角色/作品/备注 |
| 筛选 | 点击状态标签 | 列表按状态过滤 |
| 点击卡片 | 单击 | 进入收藏详情页 |
| 长按卡片 | 长按 | 进入批量选择模式 |
| 更多按钮 | 点击右上角... | 编辑/加入出物清单/标记为待出物/复制一份/删除 |
| + 按钮 | 点击 | 新增收藏页 |

**批量操作底栏**（批量模式下显示）:
- 已选择 N 件
- 加入出物清单
- 修改状态
- 删除

**API 调用**:
- `GET /api/v1/collections` (keyword, status, item_type, sort_by, sort_order, page, page_size)
- `DELETE /api/v1/collections/batch-delete` (ids)

### 2.4 新增收藏页 (`pages/collection-add/index`)

**路由**: `/pages/collection-add/index`

**表单字段**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 图片 | 图片上传 | 建议 | 1-9张，第一张为封面，支持拖动排序 |
| 名称 | 文本输入 | **必填** | 商品名称 |
| 作品 | 文本输入/历史 | 否 | 所属作品 |
| 角色 | 文本输入/历史 | 否 | 所属角色 |
| 类型 | 下拉选择 | **必填** | 吧唧/立牌/小卡/色纸/挂件/棉花娃娃/娃衣/手办潮玩/卡牌/其他 |
| 入手价 | 数字输入 | 否 | 单位：元 |
| 数量 | 数字步进 | 否 | 默认 1 |
| 入手渠道 | 输入/选择 | 否 | 闲鱼/官店/拼团/代购等 |
| 入手日期 | 日期选择 | 否 | 默认当天 |
| 当前状态 | 单选 | **必填** | 已到货/预售/待补款/待发货/待收货/待出物/已出物/不出 |
| 备注 | 多行文本 | 否 | 瑕疵、柄图等 |

**出物信息折叠区**（默认收起）:

| 字段 | 说明 |
|------|------|
| 是否准备出物 | 开关，开启后展开 |
| 出物价 | 数字输入 |
| 瑕疵说明 | 多行文本 |
| 包邮规则 | 包邮/不包邮/满额包邮 |
| 是否可小刀 | 可小刀/不刀 |
| 捆绑规则 | 文本 |

**按钮**:
- **保存**: 校验通过后保存 → 跳转到收藏详情页
- **保存并继续添加**: 保存后清空表单，保留作品/角色/类型

**保存校验**:
1. 名称不能为空
2. 类型不能为空
3. 入手价若填写需 ≥ 0

**API 调用**:
- `POST /api/v1/upload/image` (multipart/form-data, file + scene)
- `POST /api/v1/collections` (完整收藏数据)

**前端文件**:
- [collection-add/index.tsx](miniprogram-demo/src/pages/collection-add/index.tsx)

## 四、完整操作流程演练

### 4.1 主线：从零开始，生成第一份出物清单

这个是核心验证链路，按顺序走一遍即可验证所有 P0 功能。

#### 步骤 1：登录

1. 打开小程序，进入登录页
2. 勾选"我已阅读并同意《用户协议》《隐私政策》"
3. 点击"微信一键登录"
4. 授权成功 → 进入首页

> **验收点**: 登录成功后首页显示问候语，数据统计卡片，快捷操作

#### 步骤 2：新增收藏

1. 首页点击"记一件谷子"或底部"收藏"Tab → 点击"+"按钮
2. 进入新增收藏页
3. 点击上传图片 → 选择相册/拍照 → 上传 1-2 张图片
4. 填写表单：
   - 名称：**生日吧唧 A 柄**
   - 作品：**某作品**
   - 角色：**某角色**
   - 类型：选择 **吧唧/徽章**
   - 入手价：**25.50**
   - 数量：**1**
   - 入手渠道：**闲鱼**
   - 入手日期：默认今天
   - 状态：**已到货**
   - 备注：(空)
5. 点击"保存"

> **验收点**: 保存成功后跳转收藏详情页，可看到刚才填写的所有信息

#### 步骤 3：回到首页验证数据

1. 返回首页
2. 检查数据卡片：收藏 1 件
3. 检查最近收藏：显示刚添加的"生日吧唧 A 柄"

> **验收点**: 首页数据刷新准确

#### 步骤 4：新增更多收藏（模拟真实场景）

重复步骤 2，再添加 2-3 件收藏：
- 立牌、小卡、娃衣等不同类型的收藏
- 部分标记状态为"待出物"

> **验收点**: 收藏列表可正常看到所有添加的收藏

#### 步骤 5：创建出物清单

1. 从首页点击"生成出物清单"或底部"出物"Tab → "新建出物清单"
2. 进入选择收藏品页
3. 勾选想要出的 2-3 件收藏
4. 底部显示已选数量，点击"下一步"

> **验收点**: 如果未选择就点下一步，Toast 提示"请先选择至少一件收藏"

#### 步骤 6：编辑出物信息

1. 进入编辑出物信息页
2. 清单名称自动填充为"我的出物清单 + 日期"
3. 为每件商品填写出物价：
   - 生日吧唧 A 柄：出物 30 元
4. 设置为"不包邮"、"不刀"
5. 交易规则：平台选"闲鱼"，联系方式"需要请私信编号"
6. 点击"保存草稿"或"下一步：选择模板"

> **验收点**: 总价实时更新，删除商品按钮可正常移除

#### 步骤 7：选择模板

1. 进入模板选择页
2. 看到 3 个模板：简洁表格款、卡片展示款、图片墙款
3. 选择"简洁表格款"
4. 点击"生成长图"

> **验收点**: 选中模板有视觉反馈，生成中有 loading 动画

#### 步骤 8：生成结果页 - 保存图片

1. 进入生成结果页，看到长图预览
2. 点击"保存图片"
3. 授权相册权限
4. 长图保存到手机相册

> **验收点**: 保存成功后 Toast 提示"已保存到相册"；检查相册是否有带水印的长图

#### 步骤 9：验证文案

1. 在生成结果页，查看文案 Tab
2. 切换到"闲鱼文案" → 看到格式化的文案
3. 切换到"小红书文案" → 看到带 tag 的文案
4. 切换到"微博文案" → 看到微博格式
5. 切换到"微信群文案" → 看到精简文案
6. 点击"复制"按钮

> **验收点**: 每个 Tab 的文案格式不同，复制后 Toast 提示"文案已复制"

#### 步骤 10：分享验证

1. 点击"分享给好友"
2. 选择微信好友
3. 好友打开分享链接 → 看到分享页
4. 分享页显示清单图片 + 发布者昵称
5. 底部显示"我也要生成出物清单"按钮

> **验收点**: 分享页可匿名访问，显示完整信息和免责声明
